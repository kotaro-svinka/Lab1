Лабораторная работа: Обработка исключений и работа с файловой системой в Java
Данный проект представляет собой комплексную реализацию механизмов обработки исключительных ситуаций в языке Java. Работа разделена на три логических модуля, каждый из которых демонстрирует различные аспекты работы с Exception API, логированием и валидацией данных.

 Структура проекта
Задание 1: Расчет среднего арифметического (ArrayAverage.java)
Суть: Безопасная обработка числовых данных в массиве строк.
Программа проходит по массиву и суммирует числа, игнорируя текстовые вставки и ошибки индексации.
Использована структура «матрешки»: внутренний try-catch защищает итерацию цикла, а внешний try-finally гарантирует вывод финального результата.

Задание 2: Копирование файлов с обработкой ошибок (FileCopy.java)
Суть: Работа с байтовыми потоками ввода-вывода.
Функционал: Программа копирует содержимое одного файла в другой побайтово.
Обработка исключений:
- Контроль наличия исходного файла (FileNotFoundException).
- Обработка ошибок чтения/записи (IOException).
Важно: Реализовано гарантированное закрытие потоков в блоке finally, чтобы избежать блокировки файлов операционной системой.

Задание 3: Система логирования инцидентов (LabWorkTask3.java)
Суть: Генерация кастомных исключений и их регистрация в журнале.
Создано 8 пользовательских классов исключений для разных сценариев (возраст, email, стек и т.д.).
Реализован класс ExceptionLogger, который автоматически записывает детали каждого сбоя в файл log.txt с указанием времени.
Продемонстрирован механизм Exception Wrapping — перехват системной ошибки и замена её на информативное пользовательское исключение.
